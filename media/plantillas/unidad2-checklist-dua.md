# Checklist DUA para guías de aprendizaje

Unidad 2 — Actividades Auténticas e Inclusión Digital por Diseño (DUA)

Marca cada ítem que tu guía de aprendizaje cumple:

1. [ ] El contenido principal está disponible en más de un formato (texto, audio o video).
2. [ ] Existe una versión liviana / descargable para conectividad limitada.
3. [ ] Se ofrecen al menos dos formas de demostrar el aprendizaje (ej. texto, oral, visual).
4. [ ] Hay ejemplos resueltos o modelos antes de la tarea autónoma.
5. [ ] Existen pistas o andamiajes progresivos disponibles bajo demanda.
6. [ ] El lenguaje evita tecnicismos innecesarios o incluye un glosario emergente.
7. [ ] La guía explicita la relevancia o utilidad del contenido para el estudiante.
8. [ ] Las actividades permiten distintos ritmos o tiempos de entrega.

**Resultado:** cuenta cuántos ítems cumples. 7-8 = alta flexibilidad (verde). 4-6 = flexibilidad parcial (amarillo). 0-3 = guía rígida (rojo).
